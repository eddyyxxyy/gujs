Exercício 2

Crie uma função que receba dois valores inteiros como parâmetro e exiba todos os números ímpares entre estes dois números (sem incluí-los).

Minha resolução: 

      
        /**
        * Returns the odds between two values
        * @param {number} num1
        * @param {number} num2
        * @return {array}
        */
        function odds_between(num1, num2) {
            let numbers = [];
            for (let i = num1 + 1; i <= num2; i++) {
                if (i % 2 === 1) {
                    numbers.push(i);
                }
            }
            return numbers;
        }

        console.log(odds_between(3, 50))
