Exercício 3

Crie uma função que receba um valor em anos de experiência de um funcionário e retorne o nível de experiência deste conforme:

De 0 a 2 anos: Júnior
De 3 a 5 anos: Pleno
De 6 acima: Senior


Minha resolução: 

      
        /**
         * Returns the worker exp in a string
         * @param {number} years_of_work
         * @return {string}
         */
        function worker_exp(years_of_work) {
            if (years_of_work >= 0 && years_of_work <= 2) {
                return 'Junior';
            }
            else if (years_of_work > 2 && years_of_work <= 5) {
                return 'Pleno';
            }
            else if (years_of_work > 5 && years_of_work < 90) {
                return 'Senior';
            }
            return 'Invalid years of experience.';
        }
