Exercícios 1

Crie uma função que receba um objeto no formato abaixo:

      
        var cliente = {
            nome: "Maria das Graças Xuxa Meneguel",
            email: "xuxa@666.com",
            telefone: "+55 11 66666-6666",
            rua: "Rua dos Baixinhos",
            numero: 666,
            bairro: "Pinheiros",
            cidade: "São Paulo",
            uf: "SP"
        };

E retorne uma string com o seguinte formato: O(a) cliente Maria das Graças Xuxa Meneguel mora em São Paulo, SP.

Minha resolução: 

      
        function get_address_string(client) {
          return `O(a) cliente ${client.nome} mora em ${client.cidade}, ${client.uf}.`
        }
